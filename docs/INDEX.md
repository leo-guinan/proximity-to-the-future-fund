## Quick Reference

### Key Concepts
- **Proximity to the Future**: Inverse of verification latency
- **Pod**: Temporal engine (hedge fund + accelerator + startup)
- **Temporal Alpha**: Performance beyond collective time expectation
- **Time Violence**: Operational entropy (TV_t = γ_t * T_t * (1 - τ_t))

### Key Formulas
- **Foresight Efficiency**: F_t = V_t / (T_t * C_t)
- **Recursive Compounding**: F_{t+1} = F_t(1 + λ(α_t + β_t - γ_t))
- **Trust Leverage**: L_t = L_0(1 + k(τ_t - τ_eq))
- **Network Relativity**: T_B = T_A / (1 - v_rel / c_t)

### Ecosystem Layers
1. **Research Layer** (Mathlete Chain): Verifies claims
2. **Capital Layer** (Proximity Fund): Allocates capital
3. **Token Layer** ($PROX): Rewards coherence
